# Fab.Company

`Fab.Company` is an Elixir library for generating random information about a business.

## Installation

Add `:fab_company` to your list of dependencies in `mix.exs`:

```elixir
def deps do
  [
    {:fab_company, "~> 1.0"}
  ]
end
```
